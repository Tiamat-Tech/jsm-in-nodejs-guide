# The simplest template

## Features

* main entry point using `main` in `package.json`
* Uses a CJS module
* Uses ESM modules
* All ESM files use `.mjs`
* Uses `__dirname`
* Use top-level await

## Pros

The simplest possible

## Cons

* In VS Code, you can't Ctrl-Click to `add.mjs`. It also doesn't intellisense it
